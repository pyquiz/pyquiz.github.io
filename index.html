1. Keyword Argument

My function have following signature:

def myfunc(arg1, arg2, karg1='Hello', karg2=None):
    pass

Now if we change the function signature like below, will there be any problem?

def myfunc(arg1, arg2, karg1='Hello', karg3='World', karg2=None):
    pass
